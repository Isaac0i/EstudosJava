# ☕ Estudos Java

Repositório dedicado ao armazenamento das minhas práticas e experimentações em **Java**, com foco no fortalecimento da base técnica, aplicação de boas práticas e evolução contínua como desenvolvedor.

---

## 🎯 Objetivo

Este repositório funciona como um laboratório de aprendizado, onde aplico conceitos estudados e exploro diferentes abordagens para resolução de problemas.

Aqui você encontrará implementações relacionadas a:

- ✅ Lógica de programação  
- ✅ Programação Orientada a Objetos (POO)  
- ✅ Estruturas de dados e Collections (`List`, `Set`, `Map`)  
- ✅ Manipulação de datas e horários  
- ✅ Validações e regras de negócio  
- ✅ Estruturação e organização de código  
- ✅ Simulações de pequenos sistemas  

---

## 🛠 Tecnologias e Ferramentas

- **Java**
- Git & GitHub (versionamento)
- IntelliJ IDEA / VS Code
